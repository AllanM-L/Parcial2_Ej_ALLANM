El ejercicio a resolver es: Sistema de Gestión de Hospitales: Diseña un sistema usando .net y Angular para gestionar pacientes y doctores, incluyendo citas médicas. 
Tablas: Pacientes (paciente_id, nombre, apellido, fecha_nacimiento, telefono), Doctores (doctor_id, nombre, apellido, especialidad, telefono).

La funcionalidades del software es el crud respectivo de cada una de las entidades(Doctores-Pacientes-Citas) y la base de datos se desarrollo con phpMyAdmin.
